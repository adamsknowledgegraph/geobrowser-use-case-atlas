# Geo Use Cases Portfolio

A dependency-free portfolio page for Geo-powered knowledge graph examples.

Published URL:

```txt
https://adamsknowledgegraph.github.io/geobrowser-use-case-atlas/
```

## Run locally

From this folder:

```sh
python3 -m http.server 4177
```

Then open:

```txt
http://127.0.0.1:4177/
```

## Structure

- `index.html` contains the page shell and Geo-inspired copy.
- `styles.css` contains the Geo-inspired typography, colors, layout, and card design.
- `script.js` contains the use case data and renders the cards.
